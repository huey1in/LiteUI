## 安装

```bash
npm install @1ing/liteui
```

## 使用

### CDN 方式
```html
<link rel="stylesheet" href="https://unpkg.com/@1ing/liteui/dist/lite-ui.css">
<script src="https://unpkg.com/@1ing/liteui/dist/lite-ui.min.js"></script>
```
