键盘按键 Keyboard

用于显示键盘快捷键或按键提示。

基础用法

按 Ctrl + C 复制

按 Ctrl + V 粘贴

按 Esc 退出

<p>按 <kbd class="ui-kbd">Ctrl</kbd> + <kbd class="ui-kbd">C</kbd> 复制</p>
<p>按 <kbd class="ui-kbd">Ctrl</kbd> + <kbd class="ui-kbd">V</kbd> 粘贴</p>
<p>按 <kbd class="ui-kbd">Esc</kbd> 退出</p>

常用快捷键

⌘ + K
打开命令面板
⌘ + S
保存文件
⌘ + Z
撤销操作
