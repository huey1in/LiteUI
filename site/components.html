按钮 Button

按钮用于触发操作或事件。

基础按钮

<button class="ui-btn ui-btn-primary">主要按钮</button>
<button class="ui-btn ui-btn-secondary">次要按钮</button>
<button class="ui-btn ui-btn-ghost">幽灵按钮</button>

状态按钮

<button class="ui-btn ui-btn-success">成功</button>
<button class="ui-btn ui-btn-danger">危险</button>

大按钮

<button class="ui-btn ui-btn-primary ui-btn-lg">大按钮</button>

图标按钮

<button class="ui-btn-icon">
  <svg>...</svg>  <!-- 使用 Icons.bell 等 -->
</button>

<!-- 或使用 icon() 函数 -->
<script>
  btn.innerHTML = icon('bell');
</script>

API

类名说明
.ui-btn按钮基础类
.ui-btn-primary主要按钮（紫色）
.ui-btn-secondary次要按钮（灰色）
.ui-btn-ghost幽灵按钮（透明）
.ui-btn-success成功按钮（绿色）
.ui-btn-danger危险按钮（红色）
.ui-btn-lg大尺寸按钮
.ui-btn-icon图标按钮
