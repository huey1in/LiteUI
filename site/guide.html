快速开始

LiteUI 是一个轻量级组件库，零依赖，开箱即用。

CDN 引入

在 HTML 文件的 <head> 中添加：

<!-- CSS -->
<link rel="stylesheet" href="https://unpkg.com/@1ing/liteui/dist/lite-ui.min.css">

<!-- JavaScript -->
<script src="https://unpkg.com/@1ing/liteui/dist/lite-ui.min.js"></script>

<!-- 图标库 (可选) -->
<script src="https://unpkg.com/@1ing/liteui/dist/lite-icons.min.js"></script>

npm 安装

npm install LiteUI

第一个按钮

<button class="ui-btn ui-btn-primary">Hello LiteUI!</button>

暗色模式

// 切换主题
UI.theme.toggle()

// 设置主题
UI.theme.set('dark')  // 暗色
UI.theme.set('light') // 亮色
