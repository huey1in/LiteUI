开关 Switch

开关用于切换两种状态。

基础开关

<label class="ui-switch">
  <input type="checkbox">
  <div class="ui-switch-track">
    <div class="ui-switch-thumb"></div>
  </div>
</label>

带标签的开关

<label class="ui-switch">
  <input type="checkbox" checked>
  <div class="ui-switch-track"><div class="ui-switch-thumb"></div></div>
  <span class="ui-switch-label">启用通知</span>
</label>

API

类名说明
.ui-switch开关容器
.ui-switch-track开关轨道
.ui-switch-thumb开关滑块
.ui-switch-label开关标签文字
