安装方式

选择适合你项目的安装方式。

CDN 引入（推荐）

<link rel="stylesheet" href="https://unpkg.com/@1ing/liteui/dist/lite-ui.min.css">
<script src="https://unpkg.com/@1ing/liteui/dist/lite-ui.min.js"></script>
<script src="https://unpkg.com/@1ing/liteui/dist/lite-icons.min.js"></script>

npm 安装

npm install @1ing/liteui
// ESM
import '@1ing/liteui/dist/lite-ui.min.css'
import { UI } from '@1ing/liteui'
import { Icons } from '@1ing/liteui/dist/lite-icons.esm.js'

文件说明

文件说明
lite-ui.min.css压缩后的 CSS
lite-ui.min.js压缩后的 JS (UMD)
lite-ui.esm.jsES Module 格式
lite-icons.min.js图标库 (UMD)
lite-icons.esm.js图标库 (ESM)
