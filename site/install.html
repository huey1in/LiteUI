安装方式

选择适合你项目的安装方式。

CDN 引入（推荐）

<link rel="stylesheet" href="https://unpkg.com/LiteUI/dist/lite-ui.min.css">
<script src="https://unpkg.com/LiteUI/dist/lite-ui.min.js"></script>
<script src="https://unpkg.com/LiteUI/dist/lite-icons.min.js"></script>

npm 安装

npm install LiteUI
// ESM
import 'LiteUI/dist/lite-ui.min.css'
import { UI } from 'LiteUI'
import { Icons } from 'LiteUI/dist/lite-icons.esm.js'

文件说明

文件说明
lite-ui.min.css压缩后的 CSS
lite-ui.min.js压缩后的 JS (UMD)
lite-ui.esm.jsES Module 格式
lite-icons.min.js图标库 (UMD)
lite-icons.esm.js图标库 (ESM)
