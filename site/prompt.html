AI Prompt Beta

我们理解您可能在使用 AI Agent IDE，为了让您的 AI 更快学会 1inUI 的用法，我们提供了一份精简的 Prompt 文档。

获取 Prompt

下载 prompt.md 在线查看

如何使用

  1. 复制或下载 Prompt - 点击上方按钮获取完整的 Prompt 文档
  2. 发送给 AI - 将 Prompt 内容粘贴到 AI 对话中，或上传 .md 文件
  3. 开始使用 - AI 现在已经了解 1inUI，可以帮你编写组件代码了

示例对话
