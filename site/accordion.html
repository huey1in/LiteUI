手风琴 Accordion

可折叠的内容面板，用于展示分组信息。

基础用法

LiteUI 是一个轻量级的前端组件库，提供简洁优雅的 UI 组件，帮助开发者快速构建现代化的 Web 应用。
可以通过 npm 安装：npm install @1ing/liteui，或者直接使用 CDN 引入。
支持所有现代浏览器，包括 Chrome、Firefox、Safari 和 Edge 的最新版本。 
<div class="ui-accordion">
  <div class="ui-accordion-item">
    <button class="ui-accordion-header">
      标题
      <span class="ui-accordion-icon">▼</span>
    </button>
    <div class="ui-accordion-content">
      <div class="ui-accordion-body">
        内容
      </div>
    </div>
  </div>
</div>

默认展开

添加 open 类可以让手风琴项默认展开。
这个项默认是折叠状态。 
<div class="ui-accordion-item open">
  <!-- 添加 open 类默认展开 -->
</div>

JavaScript API

使用 JavaScript API 可以编程式地控制手风琴的展开和折叠。 
// 展开
UI.accordion.open(element);

// 折叠
UI.accordion.close(element);

// 切换
UI.accordion.toggle(element);

样式类

类名 说明
.ui-accordion 手风琴容器
.ui-accordion-item 手风琴项
.ui-accordion-item.open 展开状态的手风琴项
.ui-accordion-header 手风琴标题（可点击）
.ui-accordion-icon 展开/折叠图标
.ui-accordion-content 内容容器
.ui-accordion-body 内容主体
