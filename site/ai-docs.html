AI 文档

让 AI 助手通过网址学习 LiteUI 的使用方式，支持 Cursor、Claude、ChatGPT 等工具。

什么是 AI 文档？
通过提供文档网址和 sitemap，AI 工具可以自动爬取和学习 LiteUI 的完整文档，无需手动复制粘贴。这种方式更智能、更实时，文档更新后 AI 也能获取最新内容。

使用方式

方式一：提供文档网址

适用于 Cursor、Claude、ChatGPT 等支持网页访问的 AI 工具

https://LiteUI.com
使用示例
  • Cursor：设置 → Features → Docs → Add new doc，然后使用 @Docs 引用
  • Claude：直接在对话中发送网址，Claude 会自动抓取内容
  • ChatGPT：发送"请访问 https://LiteUI.com 并学习这个组件库"

方式二：提供 Sitemap

如果 AI 工具需要完整的页面索引

https://LiteUI.com/sitemap.xml

方式三：使用精简文档

如果 AI 工具不支持网页访问，可以直接复制或下载文档

使用网址的优势：
  • 文档更新后 AI 自动获取最新内容
  • 无需手动复制粘贴，更方便
  • AI 可以访问完整的示例和代码
  • 支持多页面文档的完整索引
目前支持的工具包括：
  • Cursor（通过 @Docs 功能）
  • Claude（直接发送网址）
  • ChatGPT Plus（网页浏览功能）
  • GitHub Copilot（通过配置文件）
  • 其他支持自定义知识库的 AI IDE
可以使用我们提供的精简文档（prompt.md），这是一个专门为 AI 优化的 Markdown 文件，包含了所有核心用法和示例。直接复制内容发送给 AI 即可。
