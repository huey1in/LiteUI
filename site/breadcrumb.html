面包屑 Breadcrumb

显示当前页面在系统层级结构中的位置，并能向上返回。

基础用法

首页 / 产品 / 详情 
<nav class="ui-breadcrumb">
  <a href="#" class="ui-breadcrumb-item">首页</a>
  <span class="ui-breadcrumb-sep">/</span>
  <a href="#" class="ui-breadcrumb-item">产品</a>
  <span class="ui-breadcrumb-sep">/</span>
  <span class="ui-breadcrumb-item active">详情</span>
</nav>

自定义分隔符

首页 › 文档 › 组件 
<nav class="ui-breadcrumb">
  <a href="#" class="ui-breadcrumb-item">首页</a>
  <span class="ui-breadcrumb-sep">›</span>
  <a href="#" class="ui-breadcrumb-item">文档</a>
  <span class="ui-breadcrumb-sep">›</span>
  <span class="ui-breadcrumb-item active">组件</span>
</nav>

样式类

类名 说明
ui-breadcrumb 面包屑容器
ui-breadcrumb-item 面包屑项
ui-breadcrumb-item.active 当前激活项
ui-breadcrumb-sep 分隔符
