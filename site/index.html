原生驱动，无缝集成

不需要学习复杂的 JSX 或模板语法。LiteUI 基于纯 HTML/CSS 规范，在任何框架甚至纯静态 HTML 中都能完美运行。

UI.modal.open({ title: 'Hello', content: 'World' });

深色模式

系统级的 Dark Mode 适配，无需编写一行额外的媒体查询。

状态感知

内置丰富的交互反馈组件，包括加载状态、骨架屏和 Toast 通知。

即插即用

通过简单的 Data Attributes，你可以无需编写 JS 就能初始化 Dropdown、Tabs 等复杂交互。
